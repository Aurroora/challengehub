## 1. Цель проекта
Разработка веб-сервиса для создания, отслеживания и анализа выполнения персональных челленджей. Система помогает пользователям формировать полезные привычки через ежедневные челленджи с визуализацией прогресса и мотивационной аналитикой.

## 2. Роли пользователей
- **Гость**: Просмотр доступных челленджей, регистрация в системе
- **Авторизованный пользователь**: Запуск челленджей, ежедневные отметки, создание кастомных челленджей, просмотр статистики
- **Администратор**: Управление шаблонами челленджей через админ-панель

## 3. Модели данных
### ChallengeTemplate (Шаблон челленджа)
- Поля: title, description, category, duration_days, difficulty, image, is_active, created_at
- Связи: Один ко многим с UserChallenge

### UserChallenge (Челлендж пользователя)
- Поля: user (ForeignKey), template (ForeignKey), custom_title, custom_description, start_date, status, current_streak, completed_days, notes
- Связи: ManyToOne с User, ManyToOne с ChallengeTemplate, OneToMany с DailyCheckin

### DailyCheckin (Ежедневная отметка)
- Поля: user_challenge (ForeignKey), date, is_completed, notes, rating, created_at
- Связи: ManyToOne с UserChallenge

## 4. Ключевой функционал
* Регистрация и аутентификация пользователей
* Просмотр каталога готовых челленджей с фильтрацией по категориям
* Запуск челленджей (как готовых шаблонов, так и создание своих)
* Ежедневные отметки выполнения с оценкой дня и заметками
* Визуализация прогресса в календаре
* Data Science аналитика с графиками Plotly и расчетами Pandas
* Личный кабинет с общей статистикой

## 5. Внешние интеграции / Аналитика
**Библиотеки анализа данных**: 
- Plotly 5.17.0 - для построения интерактивных графиков прогресса
- Pandas 2.0.3 - для статистических расчетов и анализа данных пользователей

**Аналитические функции**:
- Графики динамики оценок за дни
- Тепловые карты активности
- Расчет корреляций между длиной заметок и оценками
- Статистика выполнения по категориям
